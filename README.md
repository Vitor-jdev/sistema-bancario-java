#  Sistema Bancário em Java

Projeto simples de um sistema bancário feito em Java para praticar lógica de programação.

##  Funcionalidades

- Ver saldo
- Depositar dinheiro
- Sacar dinheiro (com validação de saldo)
- Sair do sistema

##  Conceitos utilizados

- if / else if / else
- while
- Scanner
- Variáveis
- Lógica de programação

##  Tecnologias

- Java
